Add a new `execute_command_streaming()` function alongside the existing `execute_command()` (keep the original for backward compatibility):

**`async def execute_command_streaming(command: str, on_line: Callable[[str], Awaitable[None]], timeout: int = 120) -> str`**

This function:
1. Creates a subprocess with `asyncio.create_subprocess_shell(command, stdout=PIPE, stderr=PIPE)`.
2. Reads stdout and stderr concurrently using `asyncio.gather()` with two `read_stream()` coroutines (avoids pipe deadlock per Pitfall 7 from research).
3. For each line read from either stream, calls `await on_line(line)` to stream it to the display in real-time. Prefix stderr lines with `[stderr] `.
4. Tracks total chars for truncation at TRUNCATION_LIMIT (10K). After the limit, still read/discard output (to avoid pipe deadlock) but stop calling `on_line`.
5. Handles timeout via `asyncio.wait_for()` — kills process on timeout.
6. Returns the same format as `execute_command()`: `"Exit code: {code}\n{output}"`.

**Inner `read_stream` coroutine:**
```python
async def read_stream(stream, prefix=""):
    nonlocal total_chars
    async for line_bytes in stream:
        line = line_bytes.decode("utf-8", errors="replace")
        total_chars += len(line)
        if total_chars <= TRUNCATION_LIMIT:
            collected.append(prefix + line)
            await on_line(prefix + line)
```

Replace the `run_agent_turn()` function with `run_agent_turn_streaming()` that uses `agent.iter()` for node-level visibility. Keep `create_agent()` and `SYSTEM_PROMPT` unchanged.

**`async def run_agent_turn_streaming(agent: Agent, prompt: str, history: ConversationHistory, display: Display) -> str`**

Implementation (following Pattern 1 from research):

```python
async with agent.iter(prompt, message_history=history.get()) as agent_run:
    async for node in agent_run:
        if Agent.is_model_request_node(node):
            # Show spinner, then stream tokens
            display.show_spinner("Thinking...")
            async with node.stream(agent_run.ctx) as request_stream:
                async for event in request_stream:
                    if isinstance(event, PartDeltaEvent) and isinstance(event.delta, TextPartDelta):
                        display.hide_spinner()  # First token: transition spinner -> stream
                        display.stream_token(event.delta.content_delta)
            display.finish_response_stream()

        elif Agent.is_call_tools_node(node):
            async with node.stream(agent_run.ctx) as tool_stream:
                async for event in tool_stream:
                    if isinstance(event, FunctionToolCallEvent):
                        # Show the command being called
                        tool_name = event.part.tool_name
                        tool_args = event.part.args
                        if isinstance(tool_args, dict) and "command" in tool_args:
                            display.show_panel(tool_args["command"], "tool_call")
                        else:
                            display.show_panel(f"{tool_name}({tool_args})", "tool_call")
                    elif isinstance(event, FunctionToolResultEvent):
                        # Tool output already streamed line-by-line via shell_tool's on_line callback
                        # But if the result has content not already shown (e.g., rejection message),
                        # display it in a tool_output panel
                        content = event.result.content
                        if isinstance(content, str) and content:
                            display.show_panel(content, "tool_output")

        elif Agent.is_end_node(node):
            pass  # Final result already streamed via model request node

history.update(agent_run.result.all_messages())
return agent_run.result.output
```
