**config.py changes:**
Add `history_path: str` field to the `Settings` dataclass, defaulting to `os.path.expanduser("~/.coding-agent/history")`. In `load_settings()`, set it from `os.getenv("HISTORY_PATH", os.path.expanduser("~/.coding-agent/history"))`.

**Create `src/codagent/input.py`:**

1. **SlashCommandCompleter** — A custom `prompt_toolkit.completion.Completer` subclass. Stores a dict of slash commands with descriptions:
   ```
   COMMANDS = {
       "/help": "Show available commands",
       "/model": "Switch the active model",
       "/approval": "Toggle approval/yolo mode",
       "/new": "Clear conversation history",
       "/exit": "Exit the agent",
   }
   ```
   In `get_completions(self, document, complete_event)`: if text starts with "/", yield `Completion` objects for matching commands with `display_meta=desc` for the fish-shell-style dropdown (per user decision). Only complete when cursor is at start of line or text only contains the slash command (don't trigger mid-sentence).

2. **Key bindings** — Create a `KeyBindings` instance:
   - Bind `escape, enter` (this is Alt+Enter / Escape then Enter) to insert a newline: `event.current_buffer.insert_text("\n")`. This gives multi-line input per user decision (Enter submits, Escape+Enter for new lines).
   - Note: Shift+Enter is not portable across terminals (Pitfall 4 from research). Escape+Enter is the reliable alternative.

3. **`create_prompt_session(history_path: str) -> PromptSession`** — Factory function that:
   - Calls `os.makedirs(os.path.dirname(history_path), exist_ok=True)` to ensure the directory exists (Pitfall 5 from research).
   - Creates and returns a `PromptSession` with:
     - `message="❯ "` (minimal symbol prompt per user decision)
     - `history=FileHistory(history_path)`
     - `completer=SlashCommandCompleter()`
     - `key_bindings=bindings`
     - `multiline=False` (Enter submits)
     - `enable_history_search=True`
     - `auto_suggest=AutoSuggestFromHistory()` (ghost text suggestions)

4. **`async def get_user_input(session: PromptSession) -> str`** — Wraps `session.prompt_async()` in a clean async function. This is what the REPL loop calls. Returns the stripped user input.

- `from prompt_toolkit.auto_suggest import AutoSuggestFromHistory`
